# 主题工厂技能

本技能提供一个精心策划的专业字体和颜色主题集合，每个主题都包含精心挑选的配色方案和字体搭配。选择主题后，可以将其应用于任何工件。

## 用途

要将一致且专业的样式应用于演示文稿幻灯片，请使用本技能。每个主题包含：
- 包含十六进制颜色代码的协调配色方案
- 标题和正文字体的互补搭配
- 适用于不同场景和受众的独特视觉识别

## 使用说明

要为幻灯片或其他工件应用样式：

1. **展示主题展示**：显示`theme-showcase.pdf`文件，让用户直观地查看所有可用主题。不要对其进行任何修改；仅展示文件以供查看。
2. **询问用户选择**：询问要将哪个主题应用于演示文稿
3. **等待用户确认**：获取关于所选主题的明确确认
4. **应用主题**：一旦选定主题，将所选主题的颜色和字体应用于演示文稿/工件

## 可用主题

以下10种主题均可用，每个主题都在`theme-showcase.pdf`中展示：

1. **深海探秘** - 专业且令人平静的海洋主题
2. **日落大道** - 温暖而充满活力的日落色彩
3. **森林树冠** - 自然且稳重大地的色调
4. **现代极简主义** - 干净且当代的灰度色彩
5. **黄金时刻** - 丰富且温暖的秋日调色板
6. **北极霜冻** - 凉爽且清新的冬季主题
7. **沙漠玫瑰** - 柔和且精致的尘色调
8. **科技创新** - 大胆且现代的科技美学
9. **植物园** - 新鲜且有机的花园色彩
10. **午夜星河** - 戏剧性且深邃的宇宙色调

## 主题详情

每个主题都在`themes/`目录中定义，包含完整的规范：
- 包含十六进制颜色代码的协调配色方案
- 标题和正文字体的互补搭配
- 适用于不同场景和受众的独特视觉识别

## 应用流程

选择偏好主题后：
1. 从`themes/`目录读取相应的主题文件
2. 在整个演示文稿中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉识别

## 创建自定义主题
如果现有主题都不适合某个工件，请创建自定义主题。根据提供的输入，生成与上述主题类似的新主题。为主题起一个类似的名称，描述字体/颜色组合所代表的含义。使用提供的任何基本描述来选择合适的颜色/字体。生成主题后，展示它以供审查和验证。之后，按照上述说明应用主题。
